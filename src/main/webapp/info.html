接口说明 1、添加一个区域

  • 请求URL

region/add

  • 请求方式

POST

  • 请求参数
一级参数二级参数必要参数类型参数说明
__param是String总参数（json字符串）
value是String区域名称
envalue否String区域英文名称
parentGuid否String父区域GUID
icon否String图标
moveRestrict是String拖动限制
indexUrl否String区域入口
remark否String备注
extdata否String扩展数据
jsonpFunc否Stringjsonp标识符
  • 请求示例
{
    " __param": {
        "type": "0", 
        "value": "南京市", 
        "envalue": "nanjing", 
        "pid": "0", 
        "icon": "", 
        "moveRestrict": "2",
        "indexUrl": "192.168.0", 
        "remark": "", 
        "extdata": ""
    }, 
    "jsonpFunc": ""
}
  • 返回参数
返回参数参数类型参数说明
resultString调用结果（true=成功，false=失败）
codeString状态码，200=成功，500=不成功，......
dataString返回给前台的数据，一个json数组字符串。
methodString后台控制器处理该操作的方法名
actionString处理该该操作的接口路径，例如 "\/region\/add"
messageString返回信息
  • 返回示例
{
    "result":true, 
    "code": 200, 
    "data": [], 
    "method": "add", 
    "action": "\/region\/add", 
    "message": "OK"
}
